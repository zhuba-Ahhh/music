# Vue3Music

## 🌈似音流年——基于Vue3.0的网易云音乐网站

开头还是感谢 [网易云音乐 NodeJS 版 API](https://binaryify.github.io/NeteaseCloudMusicApi/#/)提供的API练手。

目前已完成音乐网站的大部分功能，部分细节有可能需要优化调整。

## 💻技术栈

**Vue3 / Vue-router / ElementPlus-UI / Axios / Vuex / Vite**

## ⚡项目内容

✅ 首页

✅ 登录/退出登录

✅ 排行榜

✅ 歌单列表页

✅ 歌单详情页

✅ MV 列表页

✅ MV 详情页

✅ 歌手列表页

✅ 专辑详情页

✅ 歌曲播放

✅ 播放条（有些细节还需要修改优化）

⬜️ 评论留言点赞删除等

⬜️ 歌手详情页

⬜️ 搜索页面

⬜️ 歌曲收藏

⬜️ 个人主页

......
